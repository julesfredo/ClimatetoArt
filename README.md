# ClimateToArt
Climate to Art is a website providing the highest and lowest temperatures, in Chicago, during the summer
of the year 3 graphic artworks were exhibited.

## Features
This site features 3 Graphic Artworks from the Chicago Institute of Art's archive, querying the institutes
API for the year each artwork was exhibited.
Another API, the Meteo API is featured. Daily temperatures are retreived (using the artwork year as a variable), and the Greedy algorithm is used
to find and display the highest and lowest temperature of the Chicago summer.

## Quickstart
This site runs locally. To run this site, clone the repository with this command:

`git clone https://github.com/julesfredo/ClimatetoArt.git`

Next, navigate to the ClimatetoArt folder, and open thie ** index.html ** file in any browser.
The opening page will appear, from there use the buttons to navigate the site.
